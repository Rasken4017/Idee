# Language Learning Tool
This project aims to provide a comprehensive language learning tool that includes various features to help users learn a new language efficiently.

## Project Structure
- **/src**: Contains the source code for the application.
- **/assets**: Contains images, sounds, and other media resources for the application.
- **/tests**: Contains unit and integration tests for the project.
- **README.md**: Project documentation and structure overview.
- **requirements.txt**: List of dependencies for the project.

## Getting Started
1. Clone the repository.
2. Install the required dependencies using `pip install -r requirements.txt`.
3. Run the application using the command `python src/main.py`.
